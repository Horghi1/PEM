# PEM  - Private Expense Manager
This is a training purpose project.







# Tools & Technologies
*	Back-End: Java	
*	Front-End: FXML
*	Framework: JavaFX
*	Database: MySQL


# Functionalities
* Register new user 
*	Add new income
*	Add new expense
*	Update/Undo income/expense
*	Sort expense/income by date and type
* General income/expense overview and saving rate
*	Update/Delete account
